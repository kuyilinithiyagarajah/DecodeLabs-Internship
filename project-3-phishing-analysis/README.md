# Project 3 - Phishing Awareness Analysis

## Overview
This project simulates the role of a Cybersecurity Analyst at DecodeLabs, focused on the detection phase of social engineering attacks. The goal is to analyze phishing emails/messages, identify red flags, and build a triage framework for handling suspicious communications.

## Objectives
- Identify suspicious links, domains, and keywords in sample phishing messages
- Document red flags found across different phishing categories (mass phishing, spear phishing, whaling, smishing, vishing, quishing)
- Explain *why* each message is unsafe (sender mismatch, urgency triggers, spoofed domains, etc.)
- Build a non-expert triage checklist and decision tree (Safe / Suspicious / Malicious)

## Key Skills Demonstrated
- Threat analysis and email header inspection
- Social engineering awareness (authority, urgency, curiosity, fear/greed triggers)
- Domain spoofing detection (typosquatting, homoglyphs, combosquatting, subdomain traps)
- Security-aware decision-making (Pause, Verify, Report)
